# YO-CAB

## YO CAB is a web application for online cab booking.

### *Exciting features*
- ### 2 way chat between customer and driver.
- ### Virtual Transaction Portal
- ### Booking portal
- ### Google Maps Geolocation Api (G.P.S)
- ### Google Maps Directions Api (gives Direction)
- Bootstrap
- PHP
- HTML and CSS
- MySql
- Wamp Local Host
